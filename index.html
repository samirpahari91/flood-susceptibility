Implementation notes

important

Polygon analysis in this version reads only the raster window needed for the selected AOI. The first run may still take a moment, but it should avoid downloading the full raster before analysis.

Area totals are converted from raster cell counts using the raster resolution in feet entered in the analysis panel. Because your raster stores continuous probabilities from 0 to 1, the class totals are grouped using the probability breaks entered above.

Upload support accepts .zip shapefiles or GeoJSON. For shapefiles, package the .shp, .dbf, .shx, and .prj files together in one ZIP archive.
